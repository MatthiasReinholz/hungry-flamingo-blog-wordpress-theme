# Third-Party Notices

This theme does not bundle third-party runtime JavaScript, CSS frameworks, icon libraries, or web fonts.

## Fonts

The CSS font stacks reference Inter, Fraunces, and JetBrains Mono by name when those fonts are already available on a visitor's device. The theme does not download or ship those fonts. If bundled fonts are added later, include each font file, source, version, and license here.

## Icons

Inline SVG icons in the block template HTML are original theme assets and are distributed under the theme license.

## Screenshot

The theme screenshot is an original project asset created for this theme and is distributed under the theme license.
